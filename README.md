# See-u-behind
This project discuss about the surveillance in daily life. Cameras are present in our surroundings and serve their own purposes, yet they are often ignored. Using handtracking in touchdesigner mediapipe,finger is triggering the under layer of visual.  
Video link: https://youtube.com/shorts/t82HZLFqaPc?feature=share

# How to
1. Download the zip from files. Be aware that the toxes and the mediapipe touchdesigner should be in the same folder.
2. Check the mediapipe funtion and the camera is working.
3. Adjust the distance between you and the laptop, only between a proper distance the circle will appear.
4. Use the index finger ann thumb.The distance between two fingers is settled as the diameter of the circle. Same to the distance between index finger and the laptop.
5. Use the index finger pointing gesture as screen clearing
